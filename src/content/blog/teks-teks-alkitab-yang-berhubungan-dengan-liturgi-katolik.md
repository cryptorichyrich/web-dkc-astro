# **TEKS ALKITAB YANG BERHUBUNGAN DENGAN LITURGI KATOLIK**

  

# **Latar Belakang Alkitab untuk Ekaristi Kristen:**

  

Misa Katolik tidak hanya didasarkan pada "Perjamuan Terakhir" yang dilakukan Yesus bersama para pengikutnya, tetapi juga dipengaruhi oleh sejarah panjang perjamuan khusus yang dirayakan oleh orang-orang Yahudi kuno dan umat Kristen awal, baik sebelum, selama, dan setelah masa hidup Yesus:

  

- Abraham dan Melkisedek berbagi roti dan anggur ( [Kej 14:18-20](https://damaikasihchannel.com/kitabsuci/?q=kej14:18;kej14:19;kej14:20) )

- Abraham menyediakan makanan untuk tiga tamu ( [Kej 18:1-8](https://damaikasihchannel.com/kitabsuci/?q=kej18:1-8) )

- Orang Israel di Mesir merayakan Paskah pertama ( [Kel 12:1-28](https://damaikasihchannel.com/kitabsuci/?q=kej12:1-28) )

- Orang Israel setiap tahunnya berbagi makanan Paskah ( [Kel 12:43-51; Im 23:4-14; Bil 9:1-14; 28:16-25; Ul 16:1-8](https://damaikasihchannel.com/kitabsuci/?q=kel12:43-51;im23:4-14;bil9:1-14;bil28:16-25;ul16:1-8) )

- Bangsa Israel setiap tahun merayakan hari raya Roti Tidak Beragi ([Kel 12:14-20; 13:3-10; 23:14-15; 34:18](https://damaikasihchannel.com/kitabsuci/?q=kel12:14-20;kel13:3-10;kel23:14-15;kel34:18); dll.)

- Nabi Elisa memberi makan 100 orang dengan sedikit makanan ( [2 Raja-raja 4:42-44](https://damaikasihchannel.com/kitabsuci/?q=2raj4:42-44) )

- Yesus memberi makan 5000 orang di Galilea ( [Markus 6:30-44; Matius 14:13-21; Lukas 9:10-17; Yohanes 6:1-14](https://damaikasihchannel.com/kitabsuci/?q=mrk6:30-44;mat14:13-21;luk9:10-17;yoh6:1-14) )

- Yesus memberi makan 4000 orang lagi ( [Markus 8:1-10; Matius 15:32-39](https://damaikasihchannel.com/kitabsuci/?q=mrk6:30-44;mat15:32-39) )

- Yesus menceritakan banyak perumpamaan yang melibatkan jamuan makan dan perjamuan ( [Mat 22:1-14; 25:1-13; Luk 12:35-40; 14:15-24](https://damaikasihchannel.com/kitabsuci/?q=mat22:1-14;mat25:1-13;luk12:35-40;luk14:15-24); dll.)

- Yesus banyak makan bersama para pengikutnya dan orang lain ( [Markus 2:15-20; 14:3-9; Lukas 14:1-14](https://damaikasihchannel.com/kitabsuci/?q=mrk2:15-20;mrk14:3-9;luk14:1-14); dll.)

- Yesus berbagi Perjamuan Terakhir dengan para muridnya ( [Markus 14:12-27; Mat 26:17-30; Lukas 22:7-39; bdk. 1 Kor 11:23-25](https://damaikasihchannel.com/kitabsuci/?q=mrk14:12-27;mat26:17-30;luk22:7-39;1kor11:23-25) ​​)

- Yesus yang bangkit makan bersama dua murid di Emaus ( [Lukas 24:13-35](https://damaikasihchannel.com/kitabsuci/?q=luk24:13-35) )

- Yesus yang bangkit sarapan bersama para murid di Laut Galilea ( [Yohanes 21:1-14](https://damaikasihchannel.com/kitabsuci/?q=yoh21:1-14) )

- Umat Kristen awal di Yerusalem ikut ambil bagian dalam "Pemecahan Roti" ( [Kisah Para Rasul 2:42-47](https://damaikasihchannel.com/kitabsuci/?q=kis2:42-47) )

- Umat Kristen awal di Troas “memecah roti” bersama Paulus ( [Kisah Para Rasul 20:5-11](https://damaikasihchannel.com/kitabsuci/?q=kis20:5-11) )

- Umat Kristen awal di Korintus merayakan "Perjamuan Kudus" ( [1 Kor 10:16-17; 11:17-34; lih. Wahyu 19:9](https://damaikasihchannel.com/kitabsuci/?q=1kor10:16-17;1kor11:17-34;why19:9) )

  

# **Penggunaan Kitab Suci dalam Ibadah Kristen Awal:**

  

Sejak awal mula, ibadah Kristen tidak hanya melibatkan pembagian roti dan anggur dalam suatu jamuan ritual, tetapi juga mencakup pembacaan kitab suci (seperti yang dilakukan dan masih dilakukan dalam kebaktian sinagoge Yahudi). Umat Kristen awal tidak hanya membaca kitab suci Yahudi, tetapi segera juga menyertakan beberapa tulisan para pemimpin Kristen:

  

- Yakobus mengatakan kitab-kitab Musa telah dibacakan dengan suara keras dalam kebaktian-kebaktian di sinagoge Yahudi selama beberapa generasi ( [Kisah Para Rasul 15:21; lih. Nehemia 8:1-8; Yeremia 36:8](https://damaikasihchannel.com/kitabsuci/?q=kis15:21;neh8:1-8;yer36:8) ).

- Yesus membaca dari kitab nabi Yesaya selama kebaktian di sinagoge ( [Lukas 4:16-22](https://damaikasihchannel.com/kitabsuci/?q=luk4:16-22), mengutip [Yes 61:1-2](https://damaikasihchannel.com/kitabsuci/?q=yes61:1-2) )

- Yesus yang bangkit menjelaskan arti Kitab Suci kepada dua murid di jalan menuju Emaus ( [Lukas 24:25-47](https://damaikasihchannel.com/kitabsuci/?q=luk24:25-47) )

- Para rasul mendasarkan khotbah mereka pada kitab suci ( [Kisah Para Rasul 2:14-36; 7:2-53; 17:1-11](https://damaikasihchannel.com/kitabsuci/?q=kis2:14-36;kis7:2-53;kis17:1-11) )

- Umat Kristen membaca dari kitab-kitab “Hukum dan Kitab Para Nabi” ( [Kisah Para Rasul 13:15-44](https://damaikasihchannel.com/kitabsuci/?q=kis13:15-44) )

- Umat Kristen menyanyikan Mazmur dan himne lainnya dalam ibadah mereka ( [Kol 3:16; Ef 5:18-20](https://damaikasihchannel.com/kitabsuci/?q=kol3:16;ef5:18-20) )

- Kitab Suci dibacakan di depan umum dan diterapkan pada seluruh aspek kehidupan Kristen ( [1 Tim 4:13; 2 Tim 3:16-17](https://damaikasihchannel.com/kitabsuci/?q=1tim4:13;2tim3:16-17) )

- Surat-surat Paulus sudah dianggap sebagai “kitab suci” ( [2 Petrus 3:15-17](https://damaikasihchannel.com/kitabsuci/?q=2ptr3:15-17); bdk. [1 Tes 5:27; Kol 4:16](https://damaikasihchannel.com/kitabsuci/?q=1tes5:27;kol4:16) )
